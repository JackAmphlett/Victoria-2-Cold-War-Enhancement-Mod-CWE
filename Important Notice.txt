Important Notice:

The mod will not contain any music other than the main menu theme unless the CWE Music Addon Pack is downloaded. This is to save file space to avoid exceeding the Moddb upload size limit.

Music is now available for download at:
https://www.moddb.com/mods/victoria-2-cold-war-enhancement-mod/downloads/cwe-music-addon-pack

The music pack contains an assortment of 96 songs for a better Cold War immersion.

How to install the CWE Music Addon Pack

Copy and paste the 'music' folder from the zip file into Program Files (x86)\Steam\steamapps\common\Victoria 2\mod\CWE

When the message to "replace file" pops-up, click okay to replace the existing 'music' file in the CWE programs folder.

Enjoy!
